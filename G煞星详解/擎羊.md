**阴阳五行喜忌**
北斗 阳金 主刑伤
喜诸吉，喜四墓西北生人，喜居四墓。
忌诸凶，忌子午卯酉。
吉则权威，凶则刑伤。

**兄弟宫**
```
    1.见化忌、天刑，主争讼。见天梁，更加。

    2.遇空劫、大耗、火铃，兄弟姐妹可以倾家。不宜事业合作。
```

**夫妻宫**
```
    1.男英俊，女冷艳。

    2.独坐夫妻宫，配偶宜军警、或技术人员，或与异族通婚。

    3.独坐，配偶有一闪外伤，或动手术。

    4.见廉月桃花大耗，主离家出走。
```

**子女宫**
```
    1.主刑、主孤。与浮荡的星曜会合，主性格易流于虚诈。
    
    2.煞忌重主不孝。
```
