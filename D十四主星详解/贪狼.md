**子女宫**
```
初级讲义：
    1.贪狼会咸池天姚，主女多子少。亦主先女后子，否则偏房继室生子。

    6.紫贪同度，子必迟得。

    7.武贪同度，见左右昌曲三人。否则迟得二人。

    8.廉贪，子女二人。子则一入。

深造讲义：
    贪狼在子女宫，与桃花诸曜同会，。正室无子，但有女。庶室或外宠则生子。

    贪狼在子女宫，见吉，主父子和好；若见煞忌则父子不和。

    上述情形，可灵活运用于晚辈人际关系的推断。如正室生女，庶室生子，可表征为直接下属人手不足，需向别部门借调。

    贪狼与火铃同度，主两代感情起初融洽，至后来却冷如冰炭。并不主子女暴发。

    贪狼化忌在子女宫，亦主有外室生之子女。见煞，正室生者易小产。

    贪狼在子午，主子女四人，中有二子；见辅佐诸曜可增加。见桃花多则仅能得三女一子。子女且嫌易流为骄奢。

        （1）子午宫贪狼与蜚廉同度，见煞而无禄，主子女浪费。

**财帛宫**
```
初级讲义：
    1.贪狼守财帛宫，最喜火星或铃星同度，则主横发，或主得意外之财。

    2.亦喜禄存、化禄会照，则主财禄丰盈。

    3.若见煞再与成池、大耗、天刑、天姚、红鸾、天喜、阴煞同会，则主因争倾家。

    4.廉贞会照，亦不宜见煞，否则亦主因色而财帛耗损。

    5.会照四煞，再见空劫，主因赌博投机破家；或主染不良嗜好而至破败。

深造讲义：
    贪狼守财帛宫，手段阔绰大方，但仍精于盘算汁较，所以对于财帛的态度，其内心与外表常不一致。

    贪狼守财帛，主擅长运用金钱，但却不擅长守财，且常常有以攻为守的倾向，因此往往倾向于投机。

    贪狼与羊陀、空劫、大耗会同，则以不投机为宜，否则倾家。即使贪狼化禄，亦主波折挫败。不从事投机，则仅主事业上有极大竞争，或计划常因客观环境影响而生变化，虽受一时损失，容易东山复起。

    贪狼化权，反而增加投机性及多生变动；贪狼化禄，则无论财政状况如何，均表现阔绰，追求豪华风光的享受。唯贪狼化忌，反容易自我克制。

    火星或铃星与贪狼同度，更见禄存或化禄，决主突发或暴发。若同时有空劫者，一发即生破败。无空劫，会吉曜，然后始能保泰，且必须及时收手，不可以为永远幸运。

    贪狼叠禄，常得意外之财。但若同时与煞刑虚耗诸曜同会，则亦带有灾难性。如得伤残保险赔偿，亲属人寿保险赔偿：或有后遗症之赌博得利，均主得财而同时惹来灾祸。

    贪狼躔昌曲桃花，因色损财，更见煞刑诸曜，因色倾家，且惹官非词讼。

    贪狼与阴煞同度，主受骗；与天月同度，主因病破财；与劫煞同度，主田财起争夺。此均以同时见羊陀空劫者为然。

    贪狼化忌，与火铃同度，更见天虚，又会禄存，则其人宜以偏门起家。

    贪狼守财帛，会廉贞带禄存，又见破军，有火铃会照，亦宜财不由正路来。

    1.子午宫
        （1）子午贪狼守财帛，与紫微相对，得紫微化权拱照，则财源稳定，而可得多方面的财源。紫微化科拱照，则终生先名后利，亦主以交际应酬，多结人缘，由是营运得利。子午贪狼守财帛，与紫微相对，得紫微化权拱照，则财源稳定，而可得多方面的财源。紫微化科拱照，则终生先名后利，亦主以交际应酬，多结人缘，由是营运得利。

        （2）子宫贪狼成[泛水桃花]之局者，吉则宜经营以异性为对象之行业；凶则主因色破败。吉凶交集而同时有火铃同度，财由异性而来，亦因此生破败。

        （3）子午贪狼若不见紫微吉化，纵使贪狼化禄、化权，亦主动荡变迁，或需时时出差，或多到外埠洽谈商务。见事业宫破军与禄马同躔者，尤主外务得意(不一定迁移外地)。

        （4）子午贪狼守财帛者，以掌握财权为满足，不一定要自己拥有大量资财。故亦宜任职大企业营业部门。但不宜作计划工作。

    2.寅申宫
        （1）寅申宫贪狼独坐，与廉贞相对，一般来说，廉贞化忌，则宜半偏行业，或『凶事以求财』，以带血光为佳。在现代，经营冻肉亦属此类，亦可为外科、牙科。

        （2）倘廉贞化禄对拱，与火铃同度，亦主横发，有煞曜会合，更见阴煞天姚，则主半偏行业得财。必须见吉，然后主艺术或带装饰享受性质的行业起家。若吉煞交集，则进财同时亦有损耗，且主横发横破。

        （3）寅宫贪狼化忌，主时时改变行业。不化忌，但成『风流采杖』之局者，则主投机致祸，或经营半偏及偏门行业而致身败名裂。

    3.辰戌宫
        （1）辰戌宫贪狼独坐，与武曲相对。火铃宜与贪狼同度，若与武曲同度者，．则虽横发亦必同时起竞争。

        （2）喜武曲财星化禄，则运用财帛的能力甚强；化权、化科次之。

        （3）辰戌宫【火贪】、【铃贪】以得贪狼化禄为贵；主突然财运亨通，且略带投机色彩，但必须见好收篷，得意不宜再往。若贪狼不化禄者，突发程度较逊。

        （4）辰戌贪狼落罗网，若命宫或迁移宫有禄马交驰，则主离乡背井。以成富局。唯一生事业财源必多变动。

**疾厄宫**
```
阳木，又属水，主肝胆、内分泌系统病，又主生殖系统疾患。
初级讲义：
    1.贪狼在疾厄宫，一般均主有肝胆病。主肝旺、脾虚、胆分泌失调。幼年则主肝风抽缩。

    2.贪狼与擎羊、陀罗、火星，主有痔疮。若更会廉贞，则主肝气痛，或头晕、头风、贫血等症。

    3.贪狼与紫微同度，见煞曜，主多欲。贪狼化忌，则主手淫，并主阴亏损、虚阳上亢。中晚年则主阳痿。

    4.贪狼独坐于寅、申宫时，必与廉贞相对，主阴虚、腰痛。见煞忌刑曜，及天虚，则主亏损、肾病、生殖机能病。

    5.廉贞与贪狼同度于巳、亥宫，则主梦遗，性欲亢进。女命更主经痛、腰痛、子宫不正、赤白带、及妇女暗病。

深造讲义：
    1.凡贪狼与紫微同度或相对，与廉贞同度或相对，皆有生殖系统疾患的倾向。唯火星、铃星同度者，皆主肝病，此乃由煞曜性质影响之一例。

    2.若贪狼蹿疾厄宫，擎羊火星同躔，或陀罗铃星同躔，主脓血之疾。

    3.会廉贞化忌，女主血崩，男主遗精，重者则主阳痿或早泄。见天虚、阴煞、天伤者尤确。

    4.紫微贪狼的基本性质虽主色欲诸疾，惟以贪狼化忌者始确。所谓色欲诸疾，男主阳痿、早泄、阳亢；女主阴冷、子宫阴冷、子宫不正等，与生殖器官疾患不同。生殖器官疾患，指发炎、瘤肿之类。

    6.贪狼与廉贞对拱，有火星在三方相会，为肝肠上亢，肾水不足。

    7.贪狼与武曲对拱，金木相克，若火星与贪狼同度者，病源为阳明火盛，以致伤害肺经。若火星与武曲同度者，则为肺经本身病患，与阳明火亢无关。如肺炎、肺痨即是此类。

    8.贪狼亦主自蚀，或皮肤有白癣瘢。

    9.贪狼与文曲化忌同度，则为目疾。

    10.贪狼火星，或贪狼擎羊，主肝炎，有鼻头发红之克应。

    11.女命贪狼，见煞，亦主肝风头痛、头晕。

    12.贪狼与武曲、文曲同度，若武曲化忌，又有煞，主意外损伤，若文曲化忌，则主水厄。

    13.贪狼亦主胆病、惊风之类，以火铃同躔为确。见天刑、天月，则为肝疾，病不在胆。

**迁移宫**
```
初级讲义：
    1.贪狼为正桃花，主酒色财气。故凡贪狼在迁移宫，会化禄、化权、化科或禄存，主在异乡多酒色的应酬。多赌友、饮友，或花酒风月之友。最佳者则为有宗教信仰的人，多宗教中的朋友。

    2.若会文昌文曲，则多诗酒应酬之乐。

    3.唯与廉贞同度，则应酬固多，为人却主无事劳碌，但却无法偷闲。见煞，主因色招灾。

    4.贪狼在迁移宫会照擎羊、陀罗、大耗、空劫，主出门遭盗贼窃掠。加会咸池、天姚，主因色致祸。

    5.凡与廉贞同度而遇煞，落陷或化忌，除酒色生灾外又主遭人陷害。倘吉星祥曜同时会照，则虽有灾祸，但仍主其人能在外享乐。

    6.贪狼化忌，主为人夺爱。

    7.廉贞化忌，防脓血之灾。

深造讲义：
    1.贪狼守迁移宫，在外多应酬。应酬性质，兼视命宫、福德宫而定。此三宫见科文诸曜，则多文酒之会；见桃花诸曜，则多酒色财气；见华盖、空曜，，则多佛道宗教的朋友。

    2.贪狼化禄、化权，则应酬快乐，人缘广结。在子午二宫者，且多友人尊重。但若贪狼化忌，则每因无谓应酬而招惹麻烦、争夺。纵有吉曜同时会合，亦主多应酬而无收获。

    3.贪狼与火星、铃星同度，在外热闹，唯仅多场面上的应酬，知交难得。何禄来会·则主因应酬而得财。

    4.贪狼与火铃同度，若见煞或化忌，(煞忌齐会更劣)，则主在外仪表面风光，一有实际利益即为人夺去。

    5.贪狼与羊陀同度，主出门每多是非之挠，且往往由应酬、游宴而致。更见空劫，主遇窃盗；若加会桃花诸曜，则主因色致祸。见化忌，尤其是廉贞化忌者，情形更为严重。

    6.贪狼独坐迁移宫，化禄、化权，或与廉贞化禄同会，即使不与火铃同度；亦主迁移在外精彩发达。见火铃尤主暴发。但若同时又见煞忌，则暴发之后必暴败。

    7.贪狼在寅申二宫，与廉贞对拱，主在外酒色应酬忙碌，见吉电信局化吉者，则因之发达。见昌曲则诗酒文采，受异域人士欢迎。唯见桃花，则主酒色徵逐，桃花昌曲则为赌博。须详有无煞忌以定吉凶。

    贪狼在寅申二宫，化忌，主为人夺爱。故不宜酒色徵逐。即处理商务，亦有事夺之象。

    8.贪狼在辰与二宫，主贪恋酒色，不必更见桃花。见吉化吉曜，则经商可发。

    贪狼辰戌与同度，视为桃花，不主文采，不主文采。唯喜见魁钺，则主受人提携，或主有现成机会。

    辰戌贪狼，火铃同度，迁移可以致富，见煞则防横发横破。 [廉贞贪狼]同守迁移，主出外须历艰辛。见桃花煞忌，则酒色招祸。

    9.[紫微贪狼]居迁移，宜离出生地发展，亦利经商。

    10.[武曲贪狼]坐迁移，必为巨贾。若武曲化忌，则财色招非。

    11.流年贪狼守迁移宫，原局有煞，被流煞冲起，且见太岁、岁破、官符、大耗等曜，又逢习星会合，主遭兵贼之灾，或受贪官污吏诬枉。

**奴仆宫**
```
初级讲义：
    1.贪狼守交友宫、入庙，见左辅右弼、天魁天钺、天福、恩光、天巫会照者，主为人交游广阔，人际荚系活跃，深受友人欢迎，口碑宣传社交圈内。但实际动力则不足。

    2.若更见化禄、化杖、化科会照，则为社交界中锋头人物，受人尊敬。

    3.在上述星曜组合情形下，却主下属少实干之士，而自身亦常易忽视实干派的属员。应加注意改善，否则对自身事业有妨碍。

    4.一般情形下，贪狼守交友宫，必以嗜好结交，或书画琴棋、或医卜星相、或酒色财气、或风花雪月、或宗教信仰，视所会照星曜性质而定。最喜见天刑同度，则主为人虽多嗜好，但却能自律。亦喜见华盖同度，主喜研究哲理宗教。不宜见咸池、大耗、红鸾、天喜、天姚、陀罗、劫煞、天月，见则多酒肉朋友。更见煞曜，主因滥交生灾，或误听下属谗言而惹是生非，甚至因之破败，或遭暗算阴谋。

深造讲义：
    1.贪狼主应酬，居交友宫，基本性质已多场面上的朋友，必须见吉化吉曜，然后始主有益，及主得下属众多。否则仅为泛泛之交。

    2.若贪狼与桃花诸曜同度，又见天虚、大耗、天姚、阴煞，则主交酒肉朋友，徵歌逐色，遇事则无助力。

    3.贪狼会辅佐吉曜，又见天巫、恩光、天福，然后始主友人及下属得力。见辅佐诸曜皆为对星者，主友人及下属众多。

    4.贪狼与煞曜同度，见桃花，则主因酒色财气而惹灾非。更会化忌，主受友人或下属陷害、拖累。见空劫者，更主因此破财。

    5.贪狼与吉化同会，尤其是贪狼化禄，则主受友人欢迎，下属拥戴。然而仅限于交情，不主权威。

    6.贪狼化忌于交友宫，主受友人及下属侵夺。更见桃花、昌曲，则主因酒色起祸。

    7.贪狼在子午宫，可交年长之益友。若成『泛水桃花』，或会桃花诸曜，主与下属发生情愫。

    8.贪狼在寅申二宫，易交损友，必须见辅佐吉曜然后可免。见煞及化忌，主下属夺权。

    9.贪狼在辰戌二宫，无吉曜，下属不可靠，易生侵吞之心；见吉，下属得力。

    10.[火贪]、『铃贪』，主下属突然增加。

    11.流年贪狼在交友宫，化忌，天见煞耗，防友人或下属拖累；见白虎、官符、贯索，则固此牵连词讼。

**官禄宫**
```
初级讲义：
    1.贪狼守事业宫，宜外务工作。如营业、推销、公关等工作。必须以应酬交际为工作重点者为宜。亦可从事娱乐、演艺工作。尤以见文科诸曜，或桃花诸曜者为宜。

    2.但若于三方四正得化禄、化权、化科会照，无煞忌刑耗诸曜，而辅佐诸曜，更得天官、天福；恩光、天贵；三台、八座；台辅、封诰；天巫会照者，则可为政治圈中红人，为不同政治集团人物倚重，交际折冲，八面玲珑。

    3.若有武曲同度或会照，更见火、铃同会，则宜从事武职荣身。

    4.紫微同会，有辅曜，亦可从政。但如同时会煞，则仍以从商为宜。若地空、地劫同度，则应从事工业。唯地劫、大耗同度，则仍有风波挫折。

    5.若贪狼与火星或铃星同度，更不见其余煞曜及刑忌劫耗者，可从事投机；或主一生必有特殊幸运遭遇。

    6.若贪狼化忌，则事业发展不如理想，倘见吉曜，则主理想虽然失败，但却可获意外成就，犹之[无心插柳]。

    7.但若化忌而更见煞曜群集，则横发横破，暴起暴跌，必注意持盈保泰，始可避免。[火贪]、[铃贪]二格，可得意外成就，虽不化忌，或不见其余煞曜，但亦应克己自勉。

    8.若见煞忌，更见刑耗空劫，则易生突变而倾败。

深造讲义：
    1.贪狼守事业宫，主其事业倾向于消费、交际，或由社交而成事业。化禄、化权者，此项性质尤重，故绝对适宜于外务。

    2.贪狼与昌曲同度，最宜公开事业，或可从事带娱乐性之传播事业。

    3.若与桃花同度，则其事业以多与异性接触者为宜。否则宜从事消费，装饰或娱乐事业。

    4.贪狼如与空劫同度，须视有无昌曲，有则为艺术，无则宜从事实业。

    5.贪狼亦带艺术性质，但廉贞静态，贪狼却为动态。二者基本性质不同。

    6.在现代，贪狼亦主运动，或带韵律性之运动(如芭蕾舞)。故若贪狼化权，见煞曜而不多者，往往主运动场上竞争。

    7.贪狼与吉化、吉曜会合者，又见三台八座、台辅封诸、天官天福、恩光天贵等吉曜者，亦可从政，且主折衷调和．或为各派系势力之缓冲人物。

    8.火贪铃贪于辰戌丑未四宫，乃主持掌兵权，或在警察纪律刑法部门任职。

    9.贪狼与羊陀同茺，则以从商为宜。亦须从事消费行业或行业性质与装饰、娱乐、消费品有关。

    10.贪狼与左辅右弼同度，则主下属众多，故所经营之事业说必有相当气派。

    11.贪狼亦为饮食之星(廉贞主血，贪狼主肉)，不化忌，便主有饮食品味。见火星、天府。可从事饮食业工作。

    12.贪狼化忌在事业宫者，主凡插手事业必引起竞争。有煞曜同度者更甚，空劫同度则主因竞争而遭挫折。

    13.若贪狼化忌有吉曜同会，则理想之事业虽不能达到目的．但却别有天地，可由此创建新猷（新的谋略，指建功立业而言）。

    14.子午宫
        （1）子午贪狼与紫微对拱，主独当一面，或主自由职业。以带艺术、美观、享受色彩者为宜。

        （2）若见桃花、昌曲，则宜经营与异性有关之事业。

        （3）若见禄权科，及辅佐吉曜，则以政界为宜。唯若有羊陀会照，或擎羊同度者，为政多贪墨。

        （4）子午贪狼与天刑同度，亦主刑法，或主为艺术表演之幕后策划、编导人枕。盖台前表演须完全依照幕后之策划，故亦有纪律、原则的意味也。

    15.寅申宫
        （1）寅申宫贪狼与廉贞相对，最不宜陀罗同度。见煞(火铃亦然)，主为政贪墨，终因此致祸。唯为商暴利者，虽有挫跌之虞，却无官非。

        （2）寅申宫贪狼，以开创性行业为宜。或从事某一行业，却能别开生面，以特殊作风经营，“由此走向成功。

        （3）寅申贪狼守事业宫，时倾向于与人合作，唯多中道更张。

    16.辰戌宫贪狼对拱武曲，‘必须武曲化禄、贪狼化权，三方四正无羊陀空劫(有利刑不妨)，或有火铃会照，然后始主为军；警，刑三方面红人，独当一面，掌生杀之权。

    17.若无吉化，但有吉曜，则亦可从政，。为财赋之官(尤倾向于税务)。

    18.若有煞同度，则以从商为宜。亦必事业方向多方面发展，不只一端。

    19.倘对宫武曲化忌，从商有周转困难，从军刑恐有伤残意外。
