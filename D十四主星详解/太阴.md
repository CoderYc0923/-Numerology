9.天潢――太阴在申宫躔度。主为人福厚禄重，善应变、有权术、富幻想、有雄

10.天祥――太阴在酉宫躓度，能富能贵，唯不利婚姻。

11.天助――太阴在戌宫踴度，又称为[月照寒潭格]，主人聪明机智，感情丰富而

12.天朝――太阴在亥宫躧度，又称为[月朗天门]。主人多谋略，善计划，若化忌

初级讲义：

    （1）凡太阴人命，成其是太阴独坐，男女命皆对女亲不利。男命妨母，母死妨妻，妻死妨女；女命妨母，母死防自身（可能其实是对配偶不利），晚年妨女。日生人更甚，夜生人较佳。

    （3）太阴在命宫，主人内向，但善用心计，温和耿直，聪明机敏；男命多优秀，女命多端庄。会文昌文曲者，必有特长或专门技艺。有凤阁及天才同度，则喜书画琴棋；得桃花诸曜会照，则博学多才，流连花酒风月，亦不失为风流文士。

**财帛宫**
```
初级讲义：
    1.太阴主富，守财帛宫主富裕。

    2.入庙宫，与禄存、化禄、右辅右弼、文昌文曲相会，为富奢之命。

    3.入陷宫，有吉曜，富足；无吉曜，则须凭运限而定其财帛聚散。

    4.若会照四煞，劳心劳力而求财。否则主因财而生纠纷。

深造讲义：
    太阴为财星，但太阴主静，主藏，故在财帛上表征为储蓄、积聚，以及思考、享受。此跟天府不同，因为天府仅有守财的意义，不表征为享受及思考。这有如一个银行，银仓可能积存许多钞票，但银行职员本身并不会将存款作为自己的精神享受。太阴则等如基金经理，对因自己计划得出来的成果，自然有满足的感觉。

    太阴又与武曲不同。仍以财经机构为例，太阴等于决策投资计划的经理，而武曲则属执行计划的人员。于决策时，固然要力求冷静，在执行计划时，尤其需要当机立断。所以武曲主决断，而太阴则主计划。

    太阴之主计划，亦即擅长运用钱财。而且在精神上享受运用钱财的成果。其所得，并不全部属于自己，但仍然可因此而成富裕。

    太阴化禄，或与禄存同度，皆主富裕。化权则增加其计划理财，及享受财富的性质。化科则仅为理财的知名度。

    太阴入庙，夜生人决主富贵，其实(即社会地位)亦由富而来；日生人次之。太阴落陷，则进财维艰，夜生人较吉，曰生人更甚。

    太阴亦喜同会辅佐诸曜，以其亦为夜生人之主星也。

    太阳主贵，先有社会地位然后有钱财；太阴主富，先有钱财或在财经界上有成就，然后始有社会地位。

    太阴化忌，一般情形下主受引诱而致投资失败。所谓受诱，指受人不正确的引导，亦指受潮流引诱。若更遇文曲化忌冲会，则此种引诱且带受欺骗性质。

    若有煞曜，主成败不一且引致纠纷；见空劫，则为盗贼之灾。太阴化禄冲会文曲化忌，则主被绰或遗失。

    1.卯酉宫
        （1）卯酉宫太阴独坐，与天同相对，在酉宫入庙，故优于在卯宫落陷，唯二者皆主白手兴家。大致上可参考子午宫『天同太阴』同度时的情形推断。

        （2）卯酉太阴，在求财时常过份追求理想，因而亦时有失落感。

        （3）卯西太阴会照太阳及天梁，故当同会昌曲、天才、龙池凤阁，或且见化科之时，亦主其财源带有学术意味，一般表征为财经研究。

        （4）卯酉太阴会太阳化科，尤主学术研究。或代理名牌商品，或在有名机构服务。

    2.辰戌宫
        （1）辰戌太阴与太阳相对。一般情形下，戌宫优于辰宫。唯辰宫[日月反背]，若禄权科会，见吉曜，亦主成大富之格。

        （2）辰戌太阴最喜太阴化禄，主寓而后贵；太阳化禄则先贵而后富，故一般宜以先任职企业然后自创；会天同化禄亦主白手兴家；会天机化禄，则仅主过手钱财数量大。

        （3）若天同化忌，则劳心劳力以求财；天机化忌，则奔波劳碌；太阳化忌，则得财亦同时有是非。

        （4）太阳化禄、化科，均主有产业承继。

    3.巳亥宫
        （1）巳亥宫太阴与天机相对，亥宫优于巳寓。在亥寓者每因机遇而成巨富；在巳宫者，辛霜酸露，四处奔流。

        （3）在巳宫者，虽有相同性质，但名誉地位减等，而进财亦较少，或较艰辛。

        （4）由于天机的影响，故主变动。性质不占时，主财帛不聚，或主所任职务长期带流动色彩。若性质优良，则主所涉及者有大量金钱流通。例如从事社会工作，所牵涉者为大量救济金；从事银行业务，所牵涉者为大量存贷；从事学术研究，由专利导致大额投资，或成为公众的日常必须品。

        （5）巳亥宫太阴不宜与空劫同度，否则常生枉抛心力，或时生挫折，功败垂成。

```

    8.太阴化禄，于年限化为忌星，则主阴虚，亦可转化为肾阳不足之疾患。

**福德宫**
```

    1.太阴守福德宫，入庙，主人倾向于享乐。但所谓享乐，却不纯是物质享受，而是利用时代的物质文明，以作精神方面的享受，所以便显得高雅，或令人觉得有格调。

    3.太阴守福德宫的人，思想不倾向于竞争，更不喜横生枝节，唯有时则偏向于理想。若地空地劫同度，则主其人易流于空想而不是理想，一线之隔，相差甚远。

    4.若火星与陀罗同度，主人自寻忙碌。若擎羊与铃星同度，则其人追求理想永无满足之日，易令人多思多虑，感觉到人生疲累。

    5.太阴与天机同度，主为人心忙。但思想则时易欠深思熟虑，然而却实在喜欢清静。

    6.太阴太阳同度，最为福厚。但所谓福厚，有时却可能是人生欠积极。

    7.太阴天同同度，其人有生活情趣。

**父母宫**
```
初级讲义：
    1.太阴守父母宫，入庙，有吉曜同会者，主父母双全，无刑克伤害。

    2.若太阴落陷，则无母亲福荫，或母亲早亡或母亲离婚再嫁，自身则跟随父亲；或母亲于自己幼年时多病能对自己照料。须详会合星曜而定。一般见桃花、天刑主再嫁；见天月、天虚、阴煞则主患病，见化忌及煞刑主者死亡。

    3.凡太阴化忌于父母宫，仍须检视命盘太阳，若落陷见煞刑者主父先死。父死则母寡，亦合太阴化忌的性质

    4.太阴会四煞、天刑、化忌、阴煞、孤辰、天虚者、幼年须过继。见天马则主远离，否则有刑伤克害。

    5.太阴太阳同度，在丑宫者不利父；在未宫不利母。
