```
    1.天梁守交友宫，一般情况下主友人不多，下属不众。唯与天机同度，然后始主友人或下属众多，但仍时时哑换下属、换友人。

    3.若吉星祥曜并会，而太阳同度者，则主得交贵恩之友。且主受友人提拔。下属则多才干之士。

    4.擎羊、陀罗、火星、铃星同度或会照，主因交游生灾。故若太阳同度，无吉、见煞，不宜因友人而卷入政治圈内。亦不宜参加其竞争势力。

    5.地空、地劫同度，则主友人及下属无助力。更见大耗、劫煞、阴煞、天月，防遭受阴谋暗算，或因手下人锘浮而招致自身损失。

    6.若天梁与擎羊、天刑、大耗同度，则主与友人或下属发生词讼。
```

**官禄宫**

```
初级讲义：
    1.天梁守事业宫，以独坐午宫者格局最大，不见煞忌，得天魁天钺、文昌文曲、左辅右弼、化禄、化权、化科、禄存拱照或同度，从商从政皆为领袖人物。

    2.天梁会禄马，大利经商，主得异邦人士信任。

    3.与太阳同度，必以才艺扬名，见禄马，则名传外国。见煞，宜服务社会。

    4.天梁守事业宫，一般宜从事监察职位，亦可从事电脑、核数、设计等业。与天同会照，则为僚幕人才，从事计划工作。

    5.天梁守事业宫而见煞刑诸曜，可从事医疗工作，若煞曜多见，又见地空地劫、天刑大耗，则宜从事危险工作。否则易生灾祸，或惹官非而致破耗。

    6.天梁在巳寓者，多从事特殊行业。
```

**田宅宫**

```
初级讲义：
    1.天梁为荫星，所以居田宅宫者主得祖业。更见天巫同度，可得丰厚遗产。

    2.天梁与天机同度或会照，皆主住宅多搬迁，或房屋改建。

    3.太阳天梁同度，主因产兴讼。

    4.天梁会照禄存、化禄，凡置业必生麻烦。

    5.天梁守田宅宫者，亦主一生多住旧宅。

    6.天梁与四煞会照同度，主人家宅不安，多是非，多灾病。化忌会照，则多闲是闲非。

    7.在巳亥申宫，均不宜与地空、地劫同度，否则主人一生飘流无固定居所。

    8.天梁与天马同度，亦主飘荡。唯再与禄存会照，则主离乡背井安身。
```

**福德宫**

```
初级讲义：
    1.天梁为清贵之星，所以守福德宫时，一般均主其人精神生活重于物质生活。而且思想脱俗，不易为一般人所理解。

    2.在福德宫入庙，主其人乐天知命，胸怀旷达，喜清谈，少行动，即使行动亦一片安暇，不肯奔波忙碌。

    3.若落陷，则性质变为懒散，性喜拖延，以致误事。或误以风月为名士风流，实质思想庸常。

    4.天梁入庙，虽主观强，但做事有原则。若落陷，则原则可能变成固执。

    5.天梁在巳亥两宫，疑心极重，多思虑忧疑，以致为人欠沉稳厚重。若会煞曜，尤其是见擎羊、陀罗者，主人有心计。

    6.天梁若与天机同度，而天机化忌者，心神不宁，烦恼滋生困扰。再见陀罗同度，一生自寻烦恼；擎羊、火星、铃星同度或会照，则多是非纠纷。此皆由于固执与偏见之故。

    7.唯太阳同度，见辅弼、魁钺、恩光、天贵、台辅、封诰、天巫等会照，始主人泱泱大度，能坚守原则，且多助人之心，不生疑虑，故主心神爽朗。

    8.天梁在巳、申、亥三宫守福德，会空劫、天马，主人浮荡。

    9.天梁与禄存或化禄同会，则作事易不明朗，且喜文过饰非，又多自以为是，于是人生亦易见困扰。
```

**父母宫**

```
初级讲义：
    1.天梁化气为荫，入父母宫为祥曜。

    2.若化禄、化权、化科同度或会照，均主得父母荫庇，且主有遗产，见天巫同会者更确。唯不喜与禄存同度，主两代不和，更见煞刑等曜，则主遗产纷争。

    3.天梁落陷，一般均主刑伤克害。

    4.天梁会擎羊、天马，主人少年离家。煞重者则主再拜父母或入赘承祧。

    5.天同同度，一般不主刑伤克害。无煞，见天马，则主父子缘份浅，或父亲常离家庭，或自身远离父母。唯天同若化忌，会煞，则亦主有刑伤。或两代不和，亦宜出祀重拜。
    
    6.天同天梁同度，主幼年父母离家。

    7.太阳天梁同度，会吉者无刑伤克害。在卯宫者更主得父母福荫；在舀寓莆则父业消磨。若见煞曜，仍主刑分离，重拜父母或入赘承祧。
```
