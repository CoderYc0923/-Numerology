**迁移宫**
```
初级讲义：
    1.廉贞坐迁移宫，不宜在出生地留恋，以外走他乡为宜。喜得诸吉会照。最嫌化忌，次则嫌见煞。

    2.廉贞为次桃花，故若廉贞化忌，主在外因酒色招灾。更见煞曜、天刑、大耗于三方四正，无吉星祥曜化解，则主客死他乡。若煞刑轻，有吉曜扶持，但见天月，则主于他乡染病。

    3.廉贞最喜见紫微、天相，则可化解其凶，且主在外乡得贵人助力。

    4.廉相，得天相同度，气质转为祥和，主人出外坚忍刚毅，终能创业，尤喜见三吉化及禄存，则终有名利，即遇煞忌刑耗，亦可转危为安。但亦主在外少人缘，即得左辅右弼扶持，仍难免孤独。

    5.杀，此两星曜同度，吉凶变化甚大，有禄存、化禄会照，则主在外发达；但若会照煞星、化忌，则主在外因财生灾，或因酒色而生祸，或有血光之灾。故遇此等星曜结构居大限流年迁移宫者，不宜于此年限外出远游。若已离出生地者，则安处异邦，不再作远行。待此年限过去，始可移动。

    6.廉破，星曜结构与廉贞七杀同度者相同，故可参考前节所述。唯廉破守迁移宫者，遇吉曜，虽可发达，亦主凡事劳心费力，不如廉贞七杀者成就大。

    7.廉贪，廉贞贪狼于迁移宫同度，主人在外擅长交际，人际关系甚佳，多应酬，见吉曜，则能因此发福，见火、铃同度，更主异乡暴发，若见贪狼化忌，则凡事不如理想；若廉贞化忌，则主在外因酒色而生灾危。

    2.廉贞坐迁移宫，古代认为利于仕宦，尤其是[廉贞天相]的组合，为离乡出仕，或任职僚幕之兆。在现代，可将大企业机构的行政管理职务，视为与僚幕有同一性质。

    3.廉贞化禄，昌曲同度相会，古代认为宜离乡古藉参加典试，可中式，在原藉典试则不中式。现代亦可视为赴外邦求学的徵兆。

    11.[廉破]守迁移宫，擎羊、天刑同度或对拱，为潜伏交通意外的星系。当流年羊叠并，或廉贞化忌之年，即为出事之期，即使见吉亦主伤残。

    12.凡廉贞坐迁移，与杀、破、狼同度或对拱，文曲化忌同度，更见煞曜，主出门受骗，或主失物。
```


**官禄宫**
```
初级讲义：
    1.廉贞守事业宫，入庙，一般均宜武职，化忌者尤宜。若从事其他行业，则挫折变化甚大。有昌曲、魁钺、辅弼拱照或同度，紫微会照，则由文职而掌武权。

    2.若见武曲与文昌文曲同会，均宜从政从军，或纪律部队，主人文事武做，或武事文做。为文武兼资人才。

    3.若从商，则宜详所会照星曜而定行业。如会照贪狼，可从事设计、公关、广告。会照七杀，见天厨，可从事饮食、食品行业；会照天府，可从事金融；会照天相，则为商业策划人才或经济计划人才。昌曲同度，则从事艺术为宜。

    4.廉相，见吉星祥曜者，宜从事军警、海关工作。从政也必文事武做。见煞刑忌耗诸恶曜会合，则有撤职或失业，倒闭的危险。见煞曜轻，从商从工者，可凭技术起家。

    5.廉破，廉贞破军同守事业宫，是比较差的结构，若从商从工，均主人一生至少有一次失败与挫折，所以应详大限流年，踏实应付。若从政，则为文员之职，见吉星祥曜，亦可积资历而升迁。见煞则久处僚下。若从商，亦以血肉、饮食行业为宜，或从事带凶险性质的行业。从事军警、保安、海关工作亦可。

    6.廉府，见吉星祥曜会合者，事业成局面。由于天府保守谨慎的性格影响，不宜从事武职，可从事金融、财经行业。若见煞曜，则事业多变迁波折。见化忌，则经营不顺，不宜投机。

    7.廉贪，最宜从事交际应酬、或消费性行业，亦宜娱乐、艺术事业。见昌曲、天才同会，则以文艺工作为宜：若见咸池、天姚、红鸾、天喜，则宜从事演艺。见天马则为旅游业人才。

    8.廉杀，廉贞七杀居事业宫，最宜纪律部队出身，如军警、海关、治安单位等。若见文昌、文曲，或龙池凤阁，亦宜为技艺人才。倘更见天月，有文科二曜拱照或同度，则为外科医师、跌打医生。若有天厨同度，则可从事饮食。若廉贞化忌，虽以军警为宜，但亦可从事与血液有关的工作。

深造讲义
    1.廉贞守事业宫，古人认为属于武职，此乃由于古代社会仅重视仕途之故。在现代社会，廉贞其实亦带有浓重的商业色彩。

    2.古人论廉贞守事业宫，人庙者，武职显赫，此乃以不与煞忌同度者而言。若有煞，不宜武职，可从事工程工艺。

    3.廉贞亦为艺术之星，但倾向于土木工艺、竹木雕刻之类。在现代，则常为建筑业(不是地产业)。

    4.廉贞会文昌、文曲，主文艺，但不倾向于工艺。主对色彩有特殊敏感，故倾向于绘画、图案设计之类。

    5.廉贞守事业宫，无论何等行业，皆表现得有气派，亦可谓重外表多于实际。见煞者尤然。故最不喜空劫同会。

    6.廉贞守事业宫，武曲必在财帛宫，若昌曲入此两宫度(尤其是文曲与武曲同躔)，则主武职文做，或文职武做。在现代，此两种性质相当复杂，只能由行业分文武，再由职责任务分文武。

    7.廉贞亦主精神享受。故亦可从事关于此方面的事业。

    8.若廉贞化忌，则宜从事与血液有关的事业。一一此项性质须灵活运用。例如[血腥导演]亦属此类，由引一例，可知其性质之广泛也。

    9.以从政而言，廉贞所会的紫微有吉曜，若百官朝拱之类，则宜文职，且能独当一面；若会【紫微七杀】、[武曲七杀]，或本身即为[廉贞七杀]，则宜武职荣身。亦以煞曜不居本宫为贵。

    10.述性质，可引申入工商两业。一般而言，工属武，商属文。此为现代社会之特徵，不可拘泥于古代社会的观点。

    11.廉贞化禄于事业宫，论富裕的程度，不及廉贞化禄守命或财帛宫。因守事业宫时，有可能仅为从事外表奢华的行业。如夜总会、歌楼舞榭之类，故必须更详财帛宫的星曜以定其有财无财。

    12.凡廉贞必带出锋头、自我表现、外表浮夸的性质，或所从事的行业带此类性质，故在斗数中，廉贞不必化科而[化科]在其中矣。善于体会此意，对界定事业性质甚有帮助。

    13.子午宫
```
