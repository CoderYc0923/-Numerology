```
    14.寅申宫破军，与武曲化忌冲拱，又与火铃、空劫、天刑、天虚、大耗、阴煞等曜同会，主迁居异地至少。必有一次重大灾祸。或甚至死于异乡。

    15.[廉贞破军]不宜经商，主客缘不广。若廉贞化忌者，煞重，亦主客中发生意外。

    16.[武曲破军]见煞尚无妨，若武曲化忌则主手头困乏；会廉贞化忌，亦主灾病。

    17.凡破军守迁移宫，若不见化禄，则不利迁居异地生根，但却利流动各地以生财。

```

**奴仆宫**
```
初级讲义：
    1.破军守交友宫，一般情况下亦主因友人而招破耗，或因下属之背弃而致事业受损、财帛破散。

    2.若更见煞曜照会，则招怨背弃更甚，以致生灾涡。与擎羊、陀罗、火星、铃星、劫煞、阴煞、天刑、大耗会照，则主因友人而惹官非横祸。或受下属阴谋损害，或侵吞盗夺。

    3.最嫌武曲同度，主友人及下属口是心非。

    4.唯紫微同度最佳，主得益反，及耿直忠谏的下属吉星祥曜化解，则友人与下属仍不利，唯拖累较轻。

深造讲义：
    1.破军入交友宫，基本性质不利，．．盖化气曰耗，主耗损、怨谤二背逃毒唯破军入庙，且化禄或与禄存同度，然后始主得力。

    2.破军独坐交友宫者，主孤立。见辅佐吉曜同度，然后始得友人或下属之助力。若见辅弼，但又见煞，则终生是非口舌，且因之破耗。见空劫，虚耗者，破耗更大。

    3.破军为火铃夹，或火铃同度，更见空劫、天刑、天虚、大耗、阴煞、天月、劫煞等曜，主因友人而起官非，或遭下属盗窃、侵吞．或因之惹祸而致败耗。

    4.子午宫破军入庙，又得对宫[廉相]调和，较易招益友。若廉贞化忌，则不宜与友人合作，否则损失。更见煞刑诸曜，损耗甚大。

    5.寅申宫破军，对[武曲天相]，若有羊陀同会，则主交损友，且受拖累。若武曲化禄，或破军化禄，然后始主得助力生财。

    6.辰戌宫破军，对[紫微天相]，主得交畏友及益友，亦主得老成持重之下属。然见煞亦主助力知交易散离。

    7.凡破军坐交友富，最不宜见武曲化忌同度或拱照，主受拖累、牵连以致损失。亦不宜会贪狼化忌，见煞忌刑耗，则须防侵吞争夺。刑忌夹的天相同度或对拱，亦主受拖累或受威胁。

```

**官禄宫**
```
初级讲义：
    1.破军入庙守事业宫，适宜管理或经营工厂，或宜从事武职。

    2.若禄、权、科会，则为武职之权威人士。古人称之为[威镇华夷]。

    3.破军守事业宫，不宜经商，否则必多波折，纵有吉星祥曜会合，亦时败时成，忽起忽跌。唯破军化禄，或与禄存同度，则主其人刚毅卓拔，屡败屡起，有吉曜会照者，终抵于成。若无吉曜，则压力甚重，更会空劫，一生中必有一次倒闭或破产。若从事工业，则能藉新产品换回危机。

    4.破军守事业宫，绝不可从事投机，否则必见倾败。破军亦宜从事交通运输业，见禄马者尤佳。

深造讲义：
    1.破军守事业宫，大致略向七杀，其主要分别为七杀之开创力不如破军，然而风险亦较破军为小一一七杀守事业宫，虽必有一段时期困境，只需不投机，纵有挫折，终能东山复起，而破军则可能一蹶不振，且于顺境之时，亦必见风波，仅能安然渡过而己。

    2.以事业性质言，破军不与煞忌同躔，三方亦煞轻者，可以武职荣身。化禄、化权尤佳，更见辅佐吉曜同会，则富贵不力。

    3.若破军与煞曜同度，宜以工程技术起家。从商者必多挫折，纵能化险为夷，亦劳心伤神。

    4.破军入庙而吉煞交集者，可以经商，是则成败无常，进退之机异常重要。往往成败仅博于一线。

    5.以毅力言，破军优于七杀。故当破军守事她富时，只须连续有三四年良好的流年．即能创出事业基础。

    6.若破军化禄，则主兼行漆业，兼职兼差且往往由不以为重点的行业或职务起家。

    7.破军守事业宫，会火贪、铃贫往往于困苦艰难中突发。突发之后，若不视力亲为，或贪心无厌，则又易归于失败。

    8.破军宜于工业，其性质亦同七杀。但-七杀之特长在于管理，而破军之特长却在创新。故宜注意新产品，新技术，或从事冒险性行业。

    9.廉贞与七杀同度，宜从事工程，与破军同度，却反主处行政机构之下僚。此乃由于廉贞火可炼七杀之金，耐却与破军水火相克之故。不成格局，是故境况亦大不相同。

    10.破军喜见辅弼，则能减少辛劳。见魁铖不过多机会而已，见昌曲亦不能扭转劣境。若昌曲化习，则更易不安所业，不良不莠。

    11.破军与桃花诸曜同度，倾向于半偏行业，若更见文曲化忌者，则带巧诈性质。

    12.破军与羊陀同度，若无一技防身，则必终生不能有为，以致晚年困苦。

    13.破军亦不喜与火铃同度，主风波叠起。

    14.与空劫同度，见吉辅者，创新之能力极强，此时却又喜昌曲相济。

    15.子午宫
        （1）子午宫破军守事业宫：见吉曜亦可从政，唯若命宫有煞而无禄，则从政不免有派幂争端，而当局者却易生反覆，于年限不佳时，由此即引起极大风波。

        （2）若从工从商，则倾向于与人合作创业，且合作事业不只一端。化禄、权时尤甚。然而行业虽多，若不专，则徒然表面风光。

        （3）子午破军守事业宫，无论从事何等行业，均喜玩弄政治手腕。故必须见辅弼同会，然后始可占优势，否则每每枉作小人。

        （4）女命破军于子午守事业宫，主夫唱妇随，最宜与丈夫合作创业。

    16.寅申宫
        （1）寅申宫破军守事业宫，亦倾向于合作创业，唯若武曲化禄、贪狼化权来会，只需不会文曲化忌，则仍以独力创业为宜。

        （2）若武曲化忌，只利从工，亦有风波意外，唯风险毕竟较小。亦可从事外科、牙科等持利器以求助的行业。最不宜从商，主财不足以济其用，往往因此损失，挫败。

        （3）寅申破军主事业场而相当大，若无禄，又无吉曜，往往为外强中干。空劫同度者更甚。

        （4）亦主忌出锋头，若小有成就即胆大妄为，必为神人共弃。

    17.辰戌宫
        （1）辰戌宫破军落罗网，一生必至少有一次重大挫折，然后改行改业。

        （2）辰戌破军亦主为[副贰]必须藉别人财力或势力兴家。故最喜见魁铖。

        （3）若无吉曜，又无煞曜(或二者均见，但力量微薄者)则必须一技傍身始能生发。

        （4）有吉化吉曜，可任职财经机构。

        （5）煞重者，宜技术谋生，不宜自行经营。

        （6）辰戌破军守事业宫，不宜过份计较钱财，否则因小失大。

```

**田宅宫**
```
初级讲义：
    1.破军在辰、戌二宫守田宅，为相当理想的组合，主人既有业，亦能自置，且产业丰厚。

    2.在子、午二宫守田宅，则仅主自置丰厚。

    3.破军与紫微同度，虽产业运势不佳，但却主意外得产业。

    4.破军落陷，则主人所居住宅破旧。

    5.破军与羊、陀、火、铃、空、劫会照，主祖业凋零耗败。

    6.流年田宅宫见破军，入庙见吉，主迁新宅；落陷，见煞，则主家宅有破败。
```

**福德宫**
```
初级讲义：
    1.破军守福德宫，入庙者善于决断，因此也就少了辛劳。若破军落陷，则凡事劳心费力，且少决断，尤易生改变之心。亦正由于举棋不定，所以也就增加了人生的困扰。

    2.若武曲化忌同度，主人易决断错误，昧于形势，因此凡事便多阻碍。

    3.若廉贞化忌同度，主人[应断不断，反受其乱]，以致举措犹豫，进退失据。

    4.若无化忌同度，破军守福德宫而会四煞，则仅主易招烦恼。但举措仍嫌不安定。

    5.唯紫微同度者，始能有自我陶醉之心，精神生活反为满足，但若会照煞曜，有时亦易变为器小易盈。
```

**父母宫**
```
初级讲义：
    1.破军于父母宫亦为恶曜，性质略同七杀。

    2.最嫌武曲、廉贞同度，尤不喜化忌。唯喜紫微同度且会吉曜，则可免刑伤克害。

    3.一般情况下，破军守父母宫，均主刑伤父母，或感情不洽，无论父母存亡皆主孤独。宜远离父母，或出祀、入赘，或重拜父母。

    4.若破军与辅、弼同度，见煞刑忌者，主有病重父母。破军守父母宫，主不守祖业，或祖业耗散。
```
