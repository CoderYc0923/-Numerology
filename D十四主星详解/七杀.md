**兄弟宫**
```
初级讲义：
    1.人庙且会照禄、权、科辅弼者。兄弟众多唯有刑克。

    2.七杀守兄弟宫，而会照辅弼，或命宮会照辅弼者，亦主兄弟多而有刑克分离。感情和洽， 怔少实际助力。

    3.寅申二宫，兄弟多才清高。

    4.巳亥二宫，兄弟能贵。

    5.辰戌二宫，兄弟能富。

    6.若会照煞星化忌，仍七有灾病、刑克、分离。

深造讲义：
    1.七杀守兄弟宫，见禄、权、科及辅弼，兄弟众多。命宫见辅弼，则兄弟宫之七杀虽不会辅佐诸吉，亦主兄弟众多，但有刑克或不和。

    2.七杀守兄弟宫，煞曜聚合者，则主孤单。更见天刑乃主刑克。否则兄弟自小分离。

    七杀守财帛宫，主掌财权。这种掌财权的性质与天府不同。大致而言，天府主守，七杀主攻。天府主在现成局面下发展，七杀则主创出新局面。一一由于天府与七杀永远相对，所以虽各具不同本质，然而亦互相影响，故主守者亦有故势，主枚者亦有守势。主攻者亦有守势。彼此相成相因而不离其本质，故在其他星电会合下，天府亦可能变为以攻为守，七杀亦可以变为以守为攻。在推断时对此应仔细分别。

**奴仆宫**
```
初级讲义：
    1.七杀入交友宫，所结交者每少敦厚之士，下属亦多竞争是非。故凡值此星曜者，宜勿听闲言挑拨，凡事皆应慎重查察。兼听则明，偏听则暗。

    2.若遇左辅右弼、天魁天钺、文昌文曲，不过主结交广泛，虽争斗是非之性质得以改善，但仍主不宜。若禄存与七杀同度，则更主受人排挤倾轧。

    3.武曲化忌同度或会照，见煞曜虽轻，亦防因友人拖累，或下属倾陷而致破产。或因手下人误事而致事业失败。或如古人所言[强奴压主]。

    4.七杀守交友宫，若无吉曜化解，即不遇煞，亦主有人妒忌、陷害，或遭下属背弃，或为下属偷盗、侵吞。

    5.唯紫微同度。始可化解其不良性质，反主有助力。唯如无众吉拱照，又见煞忌刑耗，则终生受剥削侵夺拖累。

深造讲义：
    1.七杀守交友宫，主交游不广，下属稀少。若辅佐吉曜同会，然后数目始可增多，唯助力则不增。

    2.七杀必须见化禄，或会禄存，然后始可调和，则主得助力，藉以生财。

    3.唯七杀与禄存同度，则因羊陀夹的关系，又主易交小人，对自身排挤倾轧，或主受下属威胁。更与火铃、空劫、刑耗同会，则因而损失破耗。

    4.七杀与煞曜同躔，则不宜同会化权，否则主恶奴欺主。见空劫，大耗、天虚、阴煞、劫煞者主事业受挟持、侵吞。

    5.七杀与火星同度，见吉曜，主友人或下属能生极大助力，唯迟得。

    6.七杀会[火贪]、[铃贪]，主友人或下属突增，但亦易突然离散，须详所会星曜吉凶而定。

    7.七杀在卯酉二宫，武曲同度，若武曲化忌，更见煞曜，主为友人破产，或因下属而倾家，亦主受入威胁。

    8.凡七杀在交友宫，多落落寡合，故不宜与人合作事业。亦不宜听信谗言。

    9.七杀与廉贞化忌同会，然重者，主重大感情破裂，可因此招惹极大是非，若更见桃花昌曲，则牵涉男女私情。

**官禄宫**
```
初级讲义：
    1.七杀守事业寓，现代社会宜从事工业，且宜以人力资源为主者。唯绝不宜投机。因为七杀在事业宫，一生人必有一次重大挫折，若投机则难东山再起。

    2.武曲七杀同度，或廉贞七杀同度，会左辅右弼、天魁、天钺、三台八窿、天巫天福者，权威出众，武职固可荣身，若从事工业，亦主声名为外国所知。

    3.七杀不畏四煞。与四煞同度或会照，则主人于艰苦中立业，但一生仍多是非风波。

    4.七杀与地空、地劫同度，必须从事工艺，以技术立身，否则立见破败。

    5.凡七杀守事业宫，无论见吉见煞，皆不宜从事文教。因其人善于管理下属，只宜为军警或工厂领导。

深造讲义：
    1.七杀居事业宫，主管理能力强，尤以紫微七杀为然。

    2.古代无工业，最能聚众，且最能发挥管理能力者，厥为兵戎，故古人论七杀守事业宫，必许以武职荣身。现代则可为工厂管理，亦可为大企业人事部门。

    3.凡管理，必须恩威并济，七杀守事业宫者，威有馀而恩不足，[紫微七杀]尤然。故以得辅弼、魁铖、三台八座朝拱同会者为贵，得此则自能则柔相济，虽千万人皆能号令。

    4.七杀不宜与煞曜同度，火铃同会者更不宜，从事武职或人事管理，皆主生枝节，甚且有意外，或风波重重。

    5.七杀有煞曜同度，最宜一技傍身，故必需学习专门技术，不宜管理。

    6.故七杀守事业宫者，以本宫无煞始宜武职。若有煞而从事军警刑政、情报治安等业，则应居安思危。煞忌刑重者，且多意外，若会廉贞化忌，或武曲化忌，或武曲化忌，尤防有生命危险。

    7.七杀守业宫者，亦主辛劳，凡事必须亲力亲为。即使下属从多，仍难安逸。

    8.七杀与昌曲同度，以设计工作为宜。

    9.七杀与空劫同度，吉则有巧思，出人意表，凶则一生多空想，以致不良不莠。

    10.七杀与空劫同度，不宜投机，即使命宫见[火贪]、[铃贪]、亦主暴发暴败耳。

    11.七杀守事业宫者(命宫亦如之)，必须注意后天培养，尤以人和为第一，技能训练次之。缺此二者，则虽有作为，但同一星系，由于后天修养不同，影响实际命运甚大。

    12.七杀守事业宫而有煞者，见擎羊、天刑，可习外科手术、针炙、妇产科、牙科等。廉贞化忌者尤宜。

    13.若有煞而无化忌会合者，则仍以应用利器为主的行业为宜。如雕刻、理发及金属工程等。

    14.七杀与火铃同度，则以见金火的行业为佳，如冶练、烹饪、首饰之类。

    15.子午宫
        （1）子午宫七杀，与『武曲天府』相对，有煞者最宜工艺设计，主有独特概念，易为欣赏。

        （2）若辅佐诸曜同会者，则亦可主持企业或工业管理。或为军警武职。

        （3）子午火杀得禄，亦可任职财经机构，唯若见【火贪】、[铃贪]则防暴得暴失，横发横破。

        （4）子午七杀会武曲化禄，则宜武职，或财经事业，会贪狼化禄，则宜从商，唯不利投机，会破军化禄，则事业不只一端，有多方面的财源。见禄存，以任职为宜，不宜自行创业。若贪狼化忌，则一生至少有一次挫折，若武曲化忌，则唯宜任职工业；或以专门技能，计划管理等起家。

    16.寅申宫
        （1）寅申二宫，七杀与[紫微天府]相对，主工作独当一面。

        （2）会破军化禄者，一生至少有一次重要行业变动，会贪狼化禄者，从商为宜，仍不宜投机，会紫微化科，天府化科者，名大于利，以争取学历，资历为生，财帛多寡，视此而定。

        （3）若与煞曜同躔，则为工艺。

    17.辰戌宫
        （1）辰戌两宫七杀落罗网，即使投身军警两途，亦宜武职文做。

        （2）辰戌七杀守事业宫者，若经商，以周转数目巨大者为宜。若从事工业，亦必须自行销售。无论工商，皆不宜投机。

        （3）辰戌七杀守事业宫，亦一定生名大于利。

        （4）最喜见廉贞化禄、破军化权，则权利皆重。有煞亦为富局。无煞而见辅佐吉曜者，经历一段时期困滞而终成大器，是则可入武行，必能以武职荣身。

        （5）若见廉贞化忌者，以带血光事业为宜。

**福德宫**
```
初级讲义：
    1.七杀守福德宫，入庙者，品格高尚，但易流为理想太高，故虽会辅佐吉曜，仅主其人不易招嫌忌，仍因理想太高之故而少人缘。亦主妻子有刑克，妻宜迟娶、子亦宜迟。

    2.若与紫微同度，理想更高，故更易感觉人生不得志。

    3.七杀的理想偏高不同于天相。天相的理想有群体性质，例如有关整个社会制度，或公司制度，但七杀的理想则纯粹属于个人：因此天相可以说是理想主义者，而七杀则为个人主义者。

    4.最嫌与武曲同度而武曲化忌，则多是非多优虑，感到人生非常空虚。

    5.与廉贞化忌同度，个人理想受到挫折，又往往不肯面对现实，于是仍多劳碌奔走，以及心烦不宁。廉贞不化忌，则仅主忙碌，感情上不会受到挫折。

    6.七杀与羊、陀、火、铃、空、劫、大耗、天刑同度，则主人费尽心力，且奔波劳碌不安。同时人缘不佳，易招嫌忌。

    7.女命七杀坐福德宫，以迟婚或同居为宜，否则恐有刑伤。
