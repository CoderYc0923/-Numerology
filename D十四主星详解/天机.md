***“机月同梁格”***
解：天机守命宫的一种重要格局，即天机、天梁、天同、太阴在命宫的三方四正同会。
    缺点：主人有心机,好挑剔，又贪财，而多权术（喜欢在自己权力范围内弄权）。所以有一句 ***“机月同梁作吏人”***
    
    优点：主人擅机变，好谋略，长于计划，亦擅口才，所以亦可成为企业机关的僚幕（参谋），或主计划管理工作，只是不宜自己经商。

    若[机月同梁]而不弄权，则可以改变人生的际遇，所谓[公门之内好修行]（意思指在官府里当差，反而容易为百姓做点善事）

若在[机月同梁]中[天机太阴同度]，固然属于[机月同梁]，但自有其特点：
    这组星的本质是[浮动]。
    ***“天机太阴同居寅申，难免跋涉他乡”***
    这即是浮动性质的具体推断。

    凡太阴在命宫者，必须兼视[福德宫]，[机月]同度时，福德宫为巨门，而且必在辰戌两宫。这两宫是巨门最不喜之地，因而会影响到命宫的性质。

    然而[机月]又主有特别能力的机变，所以亦宜从事计划工作，若是太阴化权、化科，尤其宜金融财政的计划。
    但太阴化禄，则反主人劳心劳神，不得安宁，易患神经衰弱，且由于本质是浮动的，所以又主感情不专一。

**财帛宫**
```
初级讲义：
    1.天机守财帛宫，主有财难储。即使入庙，亦不过财来财去，无缺乏之虞。

    2.人庙，与禄存、天马、化禄会合：定富裕，财帛丰足。若天机化禄，则主由旧业中生出新业得财。单见禄存，小人觊觎。

    3.若落陷地，则财源多变化周折，且主人劳心劳力。再会煞曜，历艰辛而后得财。

    4.唯最嫌与巨门会照，必须费尽唇舌，劳神思虑然后得财，尤主凡事引起竞争。

    5.与天梁会照，则主人巧计谋财，临事多机变。

    6.最忌四煞、空劫、大耗会照，则难多发财机缘，唯聚而后散。

    7.机月，主白手兴家，或不藉父母余荫而自身创业得财。

    8.机巨，劳神费力，竞争多端。凡事未插手时，则无人注意，一旦进行，则引起别人追随争夺。且凡得财必须费尽唇舌。

    9.机梁，主机关巧计以谋求。若会煞忌，则机关赞尽亦主贫穷。

深造讲义：

    天机在财帛宫，主财帛有变动或波动。通常表现为财来去，难以积存。入庙者较佳。若落陷，则财源稍顺即生变化，或必须劳心费神，腾挪变化然后始能保住财源。

    天机化权，则性质比较稳定，然而仍必须转换财源，表征为转行转业。故以自由职业为佳；亦宜从事专业工作，则可不改行业，但改变来财之客户即可(例如律师，不必固定的客户)。

    天机为计划的星曜，在财帛宫时，亦宜倾向于劳心以求财。

    天机化禄，叠会禄存，且有吉曜，则亦可骤发。唯其骤发，必与聪明机巧有关。若化忌，则机关费尽反而财源中断，所谓『机深祸更深』是也。

    天机化禄，则气佳，但仍主财源不只一端，会吉曜，主有多方面的财路。若吉煞交集，则多反复进退。

    天机化忌，主财运稍顺即遭破坏，打击或排挤。必须于逆境时处之泰然，让时间澄清一切，表现出泱泱大度，然后始可保泰持盈，安渡逆境。

    天机化忌者，亦以从事专业为宜，且以劳心费神的行业为佳。

    天机擎羊，主劳心的程度更大。

    1.子午宫
        （1）子午宫天机入庙，必与巨门相对，故主劳心费神，破唇费舌以求财。巨门化禄者佳，化忌者劳神费舌的程度更大。

        （2）午宫天机，命宫太阴入斋；子宫天机，命宫太阴落陷，敝子宫不如午宫。在午宫者焦虑较小亦较少。在子宫者，须小情歼朗以应付横逆，且必须不用机开心计，然后始能扭转命运。

        （3）子午宫天机若与煞星同度，则主易失机缘。又或因求全求备，追求理想而反致误失良机。且一生多竞争，以致尽抛心血。

        （4）子午天机与空劫同度，主机谋失误。

        （5）子午天机化禄，或禄存同度，表征为钱财流通，故可从事经纪行业，费唇舌而能得钱财流通之利，此亦以与巨门相对之故也。

    2.丑未宫

        （2）由于与天梁相对的关系，若天梁化科，则主依靠知名度而决定财帛的多寡。天机化禄、权、科，会昌曲化科者亦然。但基天机化忌，或会昌曲化忌，则往往计谋失误，失时便应坦然承受一些失败，作为经验，以助曰后的开运。

        （3）丑未天机与天梁化禄相对，则必主机关巧计以求财。倘见阴煞、天姚、天虚同度，则计谋的性质更重，遇吉曜，可以得财，倘有煞曜同度，则主枉抛心力，不如随缘，尤以空劫同度者更甚，亦不宜独立经商。

        （2）若巨门化禄，则主得异族之财。叠见禄存，尤足以成巨富，唯必费尽心神。
```

**迁移宫**
```
初级讲义：
    1.天机为浮动的星曜，主活动，主变化。所以凡天机坐迁移宫的人，离开出生地发展反而有利。若居于出生地，反主心志动摇不定，见异思迁，且易生是非口舌。

    2.但若天机在迁移宫会照擎羊、陀罗、火星、铃星、空劫者，则出门不利，主有是非口舌、破财、惊险以及意外之灾。

    3.倘天机在迁移宫与太阴拱照，则主出门可以得财，与禄存、化禄会照者，亦主出门有财气，同会天马者更主于远方发达。一但禄马交驰而遇煞曜，则主一生奔忙，煞重者甚至流浪天涯，虽然可以得利，但劳碌奔波异常。

    4.机月，与天机、太阴对拱相会，亦主出门得财，加吉曜者尤佳。若禄马交驰，更主出门或徙移可成巨富。

    5.机巨，天机无论会照巨门，或与巨门同度，均宜出外创业。唯主费尽唇舌。但若同时会照煞曜，则主在外有是非口舌侵扰，若见化忌及天刑，则主有意外或官非。

    6.机梁，主出外有机缘，且多为现成的机会，垂手可得。或主贵人助力。见煞曜仅一两点，又见文昌文曲，则为落拓江湖的文士。

深造讲义：
    1.天机居迁移宫，主利出门离乡发展。此乃由于星曜性质浮动之故。但却不一定离乡定居，有时仅为从事经常需要离家的行业。在古代，为驿夫、水窖；在现代可为海员、航空、运输等业。其至有时仅主其人经常转工、转业。

    2.古人云：[天机出外遇贵，居家我是非。]此不可一概而论，仅于命宫无正曜，借迁移宫星曜安宫的情形下(即在申寅二宫，对宫为『天机太阴』；在卯酉二宫，对宫为[天机巨门])，若迁移寓的三方四正会吉，优于命宫，然后才主克应。

    3.又如子午宫巨门安命，会太阳化忌，且见火铃、空劫，迁移宫天机独坐，则仍以离开出生之地或久居之地为宜。尤其是天机在午得会寅宫禄存，更利迁移。

    4.迁移宫天机与火铃同度，主出外奔忙，风露辛酸，不安一地；与羊陀同度，主是非口舌；与空劫同度，主阻碍挫折。若诸煞交并，又见刑耗诸曜，则主破财或意外。尤其是当天机与天梁同度或对拱时，更主出外多困扰虚惊，但终能化解耳。

    5.迁移宫天机见禄马交驰，则虽主经常迁动，但却多享受；若见天马而不见禄存，又躔煞曜，则在外必每多劳而无功，或奔波劳累，不守一地。

    6.天机在迁移宫，与昌曲同度，不宜更见桃花，否则不但浮动不安，而且往往因情愫而羁留异地。见吉尚主在外成家，见煞则在外破耗。

    7.天机在迁移宫与诸吉同会，则大利离开出生地发展。

    8.天机与天梁同度或对拱，出门多人扶持。由于天梁为荫星，所以更主多现成机会。见魁钺同会者尤确。纵有煞星，不过虚惊。煞曜与吉曜交集，往往主因祸得福，如遇到麻烦，但却因此结识到有力的支援。

```
```
初级讲义：
```

```
    7.天机在田宅宫，与四煞、空劫、大耗同度或会照，主物业有纠纷（此物业非彼物业）。

深造讲义：
    1.天机在田宅宫，一般主产业变动，故前人认为主退去祖业，自己创置。在现代，亦主时时迁动，或时买时卖，甚或主时时变换工作环境，不一定主退祖自兴。

    2.天机之迁动是否吉利，须视所会合的星曜而定，会吉曜吉化，有时反而不主迁动，最喜见禄马，魁钺及恩光天贵、三台八座诸曜，则主迁动吉利。反之，若见火铃、空劫、化忌，则动辄得咎。

    3.流年天机化忌亦主迁动，不见煞，则亦不主凶。

    4.天机居丑未，巳亥宫垣，主居住环境复杂，或近市场，或近工厂，绝非清静。若火铃同度则更时关噪音。

    5.天机与禄存同度，有煞同会，主与邻舍不和，化忌更主受威胁或压力，见天刑、官符、白虎，则主发生诉讼，以忍让搬迁为宜，否则愈僵持愈多生是非，或主在工作机构内享受压力困扰，亦以转换工作环境为宜。

    6.天机化禄见禄马，始主迁动，而且迁动吉利，只见化禄，则为动而不动之兆。即如有迁动计划而未付诸实行。

    7.天机巨门的星曜组合，常易引起产业纠纷。亦主较难置业。或主应动不动，以致后悔。惟见禄者始主产业创新。若巨门化忌，则动辄得昝，见刑煞空劫，更主因产业或迁动，而致是非口舌连连，甚至发生官非诉讼。

    8.天机巨门有煞同度，纵见吉化亦不吉，仍主产业难守。在卯宫者较为吉利。

    9.天机太阴的星曜组合，主时时迁动，尤以寅申二宫为然，见煞则不守祖业，亦不享现成的事业，亦主时时变换工作环境。若太阴化禄，然后主易自置产业，居住环境亦较安静。

    10.天机太阴见吉化(最喜天机化禄)，又见吉曜，则买卖田宅可以获利。亦宜迁换工作环境。

    11.天机天梁的组合，基本性质主难自置产业，必四十岁以后始能置业。六十岁后产业始增加。

    12.天机天梁而天梁化禄，多住旧宅华夏。或在有年资之机构服务。

    13.天机天梁火星同度，会铃星，又见煞耗，流煞冲会，则其年主火独虚惊。

    14.天机天梁在田宅，命宫【武贪】与火星同度者，亦主时时更换住宅及办公地点，[武贪]见禄马，亦主时时迁移。

    15.一般情况下，天机守田宅宫者，受风水影响较大。仅与天梁同度，见吉化吉曜，然后下受影响。
```

**福德宫**
```
初级讲义：
    1.天机坐福德宫，一般主人思想敏锐。唯亦正因此亦令其人心绪不宁。

    2.若天机化忌，则更主人行事畏首畏尾，遇事喜钻牛角尖，以致进退失据，多虑多忧。

    3.若天机与擎羊、陀罗会照，则主人自寻烦恼，无事找事，以致终日奔波。

    4.若天机与火星、铃星、地空、地劫、天刑、大耗会照，则主人无精神享受，终日奔忙心烦，身心不得安静。

    5.若天机与吉星祥曜照会或同度，则主人心劳而身不劳：而其心劳亦不致钻入牛角尖内致生无谓困扰。但其人为了追求精神享受，或为了消磨时间，却喜学习多方面的知识与技术，然而却不喜深入。

    6.天机巨门同度，主人浮滑、多疑、善辩、思想机敏而好是非纷扰。若会吉曜则心多思虑而身安，会煞忌则劳心劳力。

    7.天机太阴同度，主其人身心劳碌，表现出来是一个喜欢热闹的人物，可是其实内心却喜清静。而且当与昌曲文曜相会，或见红鸾天喜之时，其人必喜文艺。

    8.天机与天梁同度，则虽见煞，仍主其人能忙里偷闲，寻求精神享受。但若天机化忌，则倾向于悲观消极，或时有悲天悯人的思想。

深造讲义：
    1.天机守福德宫，一般情形下主多思多虚，兴趣多端，思想不集中，考虑欠周密。若居陷宫，则更易多学不实，考虑时陷牛角尖，且多优虑。会空曜，华盖，贝旺觉人生空幻，易生宗教信仰。

    2.天机如火铃，或火铃夹者，主身劳心劳，不得安宁。化忌者更甚，且往往功败垂成。

    3.天机化禄不如化权，化权则能增加安定，减少动荡．亦使思想之周密性增加。

    4.天机会吉曜，亦不能增加其稳定，唯见煞却会增加其动荡。

    5.天机不喜巨门，无论同度或会合，皆增加其动荡。即使巨门化禄，亦仅主敏于思考，劳丽有获，若巨门化忌者，进退不安，劳心费神。

    6.天机化忌与巨门阿度或对拱，，烹枉抛力。且易常陷牛角尖而不能自拔。若与桃花诸曜同麓，则主易路感情困扰。或主爱情不专一。

    7.天机巨门与火铃同度，是为败局。化忌者多劳心，是非，化禄者圆滑，但却易自寻烦恼。亦主学不专一。

    8.天机与天梁
        （1）天机与天梁同度或对拱，无煞者，常能自得其乐，有煞，则思想易走偏锋。有空劫同度者更甚，但若沉潜于哲理，却每有成就。

        （2）天机天梁的组合，若天梁化禄，又见煞忌刑耗空劫，则多因财思虚，烦恼，或火之日夜奔波。天粱化权，化科，亦主精笔算，为计姗，管理，设计人材。

        （3）天机天梁而天机化忌，主易失眠。

        （4）天机天梁亦不喜火铃同度，主烦躁不安，易自寻烦恼。更见空曜，人生每觉空虚，因而既急躁，又消极，易成为不良不莠。

    9.天机太阴
        （1）天机太阴的组合，若不见煞，则主第六感觉强。

        （2）天机太阴，主其人既喜热闹，但离开交际场合，却有个人的天地。

        （3）天机太阴而天机化禄者，兴趣广泛，若见昌曲才，便为现代才子型人物，虽无实学，却令人觉得才气纵横。

        （4）天机太阴而太阴化禄者，主精于筹算。

        （5）天机太阴而太阴化忌，见煞刑空耗诸曜重重，每主智商低。但若吉煞交集，则却又聪明伶俐而欠沉潜。

        （6）天机太阴而天机化忌者，更见煞曜空劫，思想多不循正道。命宫若又不，则每每易染不良嗜好，以致诡诈多端。

        （7）天机太阴与文曲化忌同度，其人多偏才，更见煞曜、天姚、天虚诸曜，则善砌词，亦主工心计，若天机化科，文曲，华盖同度，则主爱好神秘事物，如术数，幻术之类。
```

**父母宫**
```
初级讲义：
    1.天机为浮荡星曜，入守父母宫，主人远离父母，否则有刑伤克害。

    2.天机守父母宫，天马同度，或在对宫拱照，主人幼年离家；年长则入赘岳家。一一在现代社会，这点意义又可转变为其人供养岳父岳母，但对亲生父母反而疏远。

    3.天机守父母宫，与擎羊、陀罗、火星、铃星、地空、地劫、天刑、化忌、阴煞、劫煞、孤辰等曜(总称[煞忌刑孤])同度或会照者，主幼年父母不全。或主重拜父母，出祀他人，或有过房父母。

    4.天机巨门同度，早年不利父母。命宫见火铃者亦主两重父母。

    5.天机太阴同度会照；或天机天梁同度会照者，不主刑克。见三化吉曜，辅佐吉曜者且主得父母荫庇。见煞忌刑孤诸曜如上述者则主父母有灾病。

    6.但天机天梁见煞忌刑孤多者，亦主父母主见极深，须小心调和代沟。

深造讲义：
    1.天机守父母富，基本性质不利父子间的人际关系。见禄权科会，然后始主稳定，否则主两代远离。入庙无煞，则远离的性质温和。

    2.天机与桃花、天马同会，主少小过继、出祀，或年长入赘，或供养岳家而不供养父母，或亲近岳家，视煞忌轻重而定。

    3.天机巨门的星系，又主父母与当局者的配偶不和，见煞尤确。

    4.天机与羊陀同会，又见空劫、天刑者，主刑克，或重拜父母。

    5.天机与火铃同度，主离不主克。

    6.天机化忌，基本性质为两代人际关系不佳。

    7.子午宫
        （1）子午天机会天同化忌，亦主关系不佳；见煞，则主有伤心落泪之事。若对巨门化忌，则童年父母不利。丑未宫者同论。

    8.丑未宫
        （1）丑未天机见吉者，主父亲有偏见。

        （2）丑未天机与空曜同度，与父母远离。

        （3）[天机太阴]为较佳组合，无煞，不主刑克分离。有煞而不会化忌，亦不主刑克，仅主分离。

        （4）[天机太阴]而天机化禄者，太阴必同时化忌，夜生人父母宫在寅宫者，早年不利父亲。

        （5）[天机太阴]天马见煞，与父母关系疏远，却亲近配偶或外室的父母。

    9.卯酉宫
        （1）卯酉宫[天机巨门]，不见煞亦主远离，或两代人际关系不佳，或不利父母。见煞者主刑克，或父业破败。然酉宫者优于卯宫。所谓[不利父母]，乃指出生后父母境遇欠佳。持此以验上司，亦主上司在工作上境况不利。此项徵验可以推断流年。

        （2）[天机天梁]在辰戌宫，见吉化、吉曜，则主有父母荫庇，亦主有上司提携。唯此荫庇提携之力不耐久。

        （3）天机天梁天马，主远离。有火星者，远离性质不吉。

        （4）[天机天梁J] 会太阳化忌，夜生人不利父亲。见煞，主出祀、过继、寄养。

    10.巳亥宫
        （1）巳亥宫天机，若躔桃花诸曜，又躔火铃，主为偏房继室所生。

        （2）巳亥宫天机，见煞，或见辅佐单星，多主两重父母。会天同化禄者尤然。

        （3）亥宫天机最不喜与巳宫太阴化忌相对，主早年父母灾病死亡。

        （4）亥宫天机借会酉宫[太阳天梁]，故见煞忌亦主不利父亲。

    11.凡天机在父母宫，稍见刑忌即有事端，应详各流年父母宫而厘定其克应。克应之期，多在太阴太阳的宫度。
```
