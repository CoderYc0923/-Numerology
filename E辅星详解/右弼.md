**阴阳五行喜忌**
中天 阴水 主助力 司制令
喜诸占不喜诸凶。忌居卯酉，余宫皆占。
吉则助力，凶则掠夺。

阴水，主肾弱。

**兄弟宫**
```
    1.见武曲七杀，见煞，主有刑克。如父母宫见桃花诸曜，则主异胞。空劫、化忌、煞曜多见，孤单。

```

**夫妻宫**
```
    1.带桃花，主第三者介入；独坐尤的。

    2.主两女共事一夫。见机月尤的。

    3.见忌空劫，主离。
```

**子女宫**
```
    1.主子女得力，可以兴家。

    2.亦主子女数目比正曜星系所显示者为多。
```

**财帛宫**
```
    1.主得人助力，或得人提拔推崇而生财。
```

**迁移宫**
```
    1.遇吉曜则主多助力。

    2.加煞曜刑忌，则反主小人竞争，是非口舌。
```

**奴仆宫**
```
    1.一般主友人下属众多，且得力。唯必须遇吉曜会合者始是。

    2.若与天虚、阴煞、天月会合，则虽多无益。宜远离小人。
```
